# Hyperspectral Change Detection

## Overview
Detects environmental and land-cover changes from hyperspectral satellite imagery using Hybrid Quantum Computing techniques.

## Technologies
- Python
- NumPy
- Qiskit
- OpenCV
- Matplotlib

## Input
Hyperspectral satellite images.

## Output
Detected change map.

## Folder Structure

modules/
config.py
main.py
requirements.txt

## 👥 Project Information

HQC-Vision is a collaborative group project focused on applying Hybrid Quantum Computing techniques to Computer Vision for Industrial Inspection, Medical Imaging, and Hyperspectral Remote Sensing.

This repository is maintained and documented by Sai Teja Gunda for project organization, documentation, and future development.
