# Microcalcification Detection

## Overview
Detects microcalcifications from mammogram images using classical preprocessing and Hybrid Quantum Computing techniques.

## Technologies
- Python
- OpenCV
- NumPy
- Qiskit
- Matplotlib

## Input
Mammogram images.

## Output
Detected microcalcification regions.
